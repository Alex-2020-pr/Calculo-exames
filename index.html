Guia de Referência

Tamanhos Típicos por Modalidade

Exames Simples (10-50 MB):
  • Raio-X Digital: 10-30 MB
  • Densitometria Óssea: 15-25 MB
  • ECG: 2-5 MB
  • Odonto Panorâmico: 20-40 MB
Exames Moderados (50-300 MB):
  • Ultrassom: 40-100 MB
  • Mamografia: 100-200 MB
  • Tomografia (CT) simples: 150-300 MB
  • CBCT Odontológico: 200-400 MB
Exames Complexos (300+ MB):
  • Ressonância Magnética: 300-1000 MB
  • Tomografia com contraste: 400-800 MB
  • Tomossíntese Mamária: 500-1000 MB
  • Endoscopia com vídeo: 800-1500 MB
  • Hemodinâmica/Angio: 1500-3000 MB
Nota: Esses valores são estimativas. O tamanho real pode variar significativamente dependendo da resolução, número de cortes, uso de contraste e formato de arquivo (DICOM, JPEG, MPEG, etc.).

Informações sobre Custos:

Projeção de Custos

Fatores que Influenciam Custos

Tipos de Armazenamento:
  • Hot Storage: Acesso rápido (R$ 0.10-0.20/GB) - Para exames recentes
  • Cool Storage: Acesso menos frequente (R$ 0.05-0.10/GB) - Para arquivamento
  • Cold Storage: Arquivo de longo prazo (R$ 0.01-0.05/GB) - Para retenção legal
Provedores Populares:
  • AWS S3: R$ 0.12-0.25/GB (Standard)
  • Google Cloud: R$ 0.10-0.20/GB (Standard)
  • Azure Blob: R$ 0.11-0.22/GB (Hot)
  • Wasabi/Backblaze: R$ 0.03-0.06/GB (Econômico)
Estratégias de Otimização:
  • Implementar tiered storage (misturar hot/cold)
  • Comprimir arquivos DICOM (lossless)
  • Definir políticas de retenção claras
  • Considerar armazenamento híbrido (cloud + on-premise)
Nota: Esta ferramenta fornece estimativas baseadas em valores médios do setor. Os custos reais podem variar conforme o provedor de nuvem, configurações específicas e variações regionais. Recomenda-se consultar diretamente os fornecedores para orçamentos precisos.
